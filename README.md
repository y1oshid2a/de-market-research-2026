# DE Market Research 2026

A structured research project analyzing **Data Engineer job postings** across major Japanese tech companies (mega-ventures) and global IT firms — built as a portfolio artifact to demonstrate systematic research, analytical thinking, and modern engineering workflow practices.

> This research was conducted using Claude Code (CLI AI agent) to demonstrate modern AI-driven engineering workflows.

---

## What This Repository Shows

| Dimension | Detail |
|-----------|--------|
| **Domain knowledge** | Deep understanding of DE roles, tech stacks, and hiring trends across Japanese and global markets |
| **Analytical rigor** | Structured comparison across skills, cloud platforms, compensation, and work conditions |
| **Workflow** | AI-assisted research pipeline using Claude Code — prompt design, iterative refinement, structured output |
| **Documentation** | Consistently formatted Markdown, templates, and versioned analysis |

## Workflow

This project uses **Claude Code** as a CLI AI agent to assist with:

- Structuring raw job description text into normalized Markdown records
- Drafting and iterating analysis summaries
- Maintaining consistent documentation conventions across all entries

The goal is not just to surface hiring trends, but to demonstrate how AI tooling can be integrated into everyday engineering and research workflows — a skill increasingly relevant to modern data teams.
